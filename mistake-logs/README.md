# Mistake Logs

This folder holds mistake-log entries produced during `dsa-tutor`
sessions, using the format and category taxonomy defined in
[`skills/dsa-tutor/SKILL.md`](../skills/dsa-tutor/SKILL.md).

An entry is added only once a learner has confirmed the actual root
cause of a mistake — the tutor does not invent one. Isolated typos,
fatigue slips, and one-off syntax errors with no conceptual weight
are not logged.

Categories in use:

- `reasoning-error`
- `pattern-recognition-error`
- `implementation-error`
- `reading-error`
- `math-error`
- `off-by-one`
- `state-tracking-error`

This folder is currently empty. Entries accumulate here as real
tutoring sessions produce learner-confirmed root causes.
